# Email Setup (Make.com)

Option 1: Gmail
- Use "Gmail: Send an email" module.
- Connect your Gmail account.
- Be careful with sending limits and reputation.

Option 2: SMTP / SendGrid / Mailgun
- Use "Email: Send an email" with SMTP.
- Recommended for larger campaigns.

Always:
- Use a custom domain email (e.g., yourname@yourdomain.com).
- Include an unsubscribe or "reply STOP" style instruction.
